
# Cybersecurity Home Lab (VM Environment)

This repository contains my hands-on cybersecurity experiments and projects performed in a virtual machine (VM) lab environment.

I use this lab to simulate real-world systems and practice both defensive and offensive security concepts in a controlled setup.

---

##  Purpose
The goal of this home lab is to develop practical cybersecurity skills through real system interaction, including:

- Operating system and network exploration in a VM environment  
- Security testing and vulnerability analysis  
- Log analysis and system monitoring  
- Basic offensive and defensive security techniques  
- Tool usage in realistic lab conditions  

---

## Lab Environment
- Virtual Machine (VirtualBox / VMware)
- Kali Linux / Ubuntu / Windows (depending on test scenario)
- Networking setups (NAT / Host-only / Bridged)
- Security tools like:
  - Wireshark
  - Nmap
  - Netcat
  - metasploit
  - john the ripper
  - hydra


---

##  Structure

Each folder represents a separate lab or experiment:

- `/defence` → Network scanning, traffic analysis, packet analysis   
- `/offence' → network scanning, ennumeration, exploitation

All activities are performed in a controlled virtual environment for educational purposes only.
